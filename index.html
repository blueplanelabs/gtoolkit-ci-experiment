This is a testing page in a Lepiter database to try the automatic code generation by a CI workflow.

The following snippets containing Pharo code should be automatically applied by the building workflow of the project in order to generate a new Smalltalk image containing the corresponding generated code.

For instance, the following snippet will create programatically a new empty Smalltalk class named LepiterDefinedClass that inherits from Object in the category LepiterDefinitions: 

Object subclass: #LepiterDefinedClass 
    instanceVariableNames: '' 
    classVariableNames: '' 
    category: 'LepiterDefinitions'

The following snippet add a simple method to the previously defined class: 

LepiterDefinedClass compile: '
warningColor

    ^ Color orange
'

Finally we can define an example to test the expected behaviour: 

LepiterDefinedClass compile: '
exampleWarningColor
    <gtExample>
    
    |warning|
    warning := self warningColor.
    self assert: warning equals: Color orange.
    ^ warning
'
